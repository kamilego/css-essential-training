Domes generator

This script is over 80% of my masters thesis. Main goal is to automate sensitivity analysis of domes constructions by creating many files with modified vertices of hemispherical domes. User can by own set height, cross-section, material and loads. Due this solution it is easy to find right size of dome were possible is to show the power of sensitivity analysis.

GitHub Link

Antennas rotate solver

Script automates boring and monotonous task which is to setup antennas with right azimuth in AutoCAD program. User sets values of degrees and types of antennas. Python takes control of mouse and keyboard and in a short time does all donkey work. It helps a lot and saves a lot of time.

GitHub Link

Forces analysis

Scripts help with analysis of forces/displacements of existing lattice mast which are overload. Autodesk Robot program provides FEM calculations and returns as a result csv file that contains over 200 thousand lines of data. Main goal is to sort and manipulate data with certain way to save them in excel file.
